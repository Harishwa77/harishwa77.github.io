
# EchelonAI | Project Manifest & AI System Prompt

## Core Identity
EchelonAI is an **Advanced AI Startup Ecosystem Engine** designed for Founders, Investors, and Talent. It leverages Genkit (Gemini 1.5/2.0) and live data layers to provide professional-grade strategic intelligence.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **AI/GenAI**: Genkit for Google AI (Gemini 2.5 Flash / Flash-Image / Imagen 4)
- **Database**: Firebase Firestore (Real-time pool & private reports)
- **Auth**: Firebase Authentication (Anonymous & Secure)
- **Integrations**: 
  - **NewsAPI**: Real-time industry sentiment analysis.
  - **Alpha Vantage**: Financial benchmarks and sector performance.
  - **Google Maps JS API**: Geospatial Intelligence for regional strategy.
- **UI/UX**: Tailwind CSS, ShadCN UI, Lucide Icons, Recharts (Data Visualization).

## Backend Architecture
The backend is serverless and event-driven, optimized for high-speed AI inference and real-time data synchronization.

1. **Persistence Layer (Firestore)**:
   - `/users/{userId}`: User profiles with `userType` and metadata.
   - `/startups_for_investment/{startupId}`: The "Ecosystem Pool" where founders publish vetted ideas.
   - `/users/{userId}/analysisReports/{reportId}`: Private, stateful strategic reports generated by the engine.

2. **Logic Layer (Genkit & Server Actions)**:
   - **Genkit Flows**: Encapsulated AI logic for roadmap generation, investment analysis, and talent matching.
   - **Server Actions**: Secure entry points for the frontend to trigger AI flows and database mutations.

3. **Intelligence Layer (External APIs)**:
   - Middleware services fetch real-time sector data (Alpha Vantage) and global news (NewsAPI) to ground AI responses in current reality.

## Functional Modes
1. **Founder Mode**: Roadmap generation, Geospatial Intelligence, and API stack recommendations.
2. **Investor Mode**: 5-year growth projections, ROI simulations, and risk assessments.
3. **Talent Mode**: Semantic matching between user skills and the real Firestore startup pool.
4. **Evolution Mode**: Genetic mutation engine creating 3 high-fitness startup variants.
